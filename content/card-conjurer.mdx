What's Card Conjurer?
Card Conjurer is a free, web-based custom card creator that empowers users to create personalized game cards using provided templates, or even custom templates. On Card Conjurer, you can make anything! Whether you're a custom card enthusiast, game designer, or just want to have some fun, Card Conjurer is the right tool for you!

We offer several free card templates that you can start using right away, and using the template creator, you can even build your own. What's more, is that users who sign in can upload their custom templates, and set them to public so that anyone with the link can use their custom templates, too.
